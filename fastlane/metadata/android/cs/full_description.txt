Co lze přemapovat?

 * Gesta otisků prstů na podporovaných zařízeních.
 * Tlačítka hlasitosti.
 * Navigační tlačítka.
 * Bluetooth/kabelová klávesnice.
 * Tlačítka na ostatních připojených zařízeních by měla také fungovat.

POUZE HARDWARE tlačítka mohou být znovu mapována.
Žádné z těchto tlačítek nebude fungovat a tato aplikace NENÍ navržena pro kontrolu her. OEM/dodavatel vašeho zařízení může zabránit jejich přemapování.

Můžete kombinovat více klíčů z konkrétního zařízení nebo jakéhokoli zařízení a vytvořit "trigger". Each trigger can have multiple actions. The keys can be set to be pressed at the same time or one after another in a sequence. Keys can be remapped when they are short pressed, long pressed or double pressed. A keymap can have a set of "constraints" so it only has an effect in certain situations.

What can’t be remapped?
 * Power button
 * Bixby button
 * Mouse buttons
 * Dpad, thumb sticks or triggers on game controllers

Your key maps don't work if the screen is OFF. This is a limitation in Android. There is nothing the dev can do.

Co ostatní věci může aplikace dělat?
 * Možnost zobrazit trvalé oznámení, které může pozastavit/obnovit vaše klíčové mapy.
 * Automaticky změnit klávesnici na mapu kláves a/nebo zobrazit výběr klávesnice, když je vybrané zařízení Bluetooth připojeno a přepnout zpět na staré zařízení po odpojení.
 * Rychlé nastavení pro pozastavení/obnovení klíčových map a přepnutí klávesové mapy.
 * Vytvořte widget domovské obrazovky pro provedení výše uvedených akcí.
